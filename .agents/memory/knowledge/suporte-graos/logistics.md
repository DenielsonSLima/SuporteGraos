# Logística e Carregamentos — Suporte Grãos ERP

## Logística
- **Gestão de Fretes**: Controle por transportadora, placa e motorista.
- **Valor em Risco**: Valor total das cargas em trânsito.
- **Base de Cálculo**: Toggle entre BASE ORIGEM / BASE DESTINO para cálculo do frete.

## Gestão de Carga (Modal)
- **Confirmar Pesagem**: Gatilho crítico que gera contas a receber e atualiza status para "Descarregado".
- **Custo Logístico**: Preço por TON.

## Sub-módulos (Compra)
- **Romaneios**: Cada carga tem data, trans., mot., pesos O/D.
- **Adiantamentos**: Pagamentos antecipados que devem ser abatidos no saldo final.
- **Despesas Extras**: "Debitar do Produtor" (reduz saldo) ou "Custo Empresa" (acrescenta ao custo).
